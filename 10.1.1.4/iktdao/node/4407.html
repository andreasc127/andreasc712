日期: 國曆中華民國98年(2009)7月18日 農曆中華民國九十八年歲次己丑閏五月十八日
主題: 先齊其家
道場: 發一慈法
班別: 講師班
佛堂: 明順壇
臨壇仙佛：南極老壽星 中八仙呂洞賓 靈隱禪師瘋瘋和尚 中八仙藍采和

西元二００九年歲次己丑閏五月廿六日　　　　　　慈訓

　南極壽星慈訓：
修道路回家的指引　人生苦體悟道修行　每個人都有潛在的才能　能克服而前奔　流過淚心境能提升　有過痛才能更上層　一樣的天空　不同的人生　有一樣的感動　請不要自怨自艾心惶恐　心踏實穩重　只要向前衝　告訴自己天生我材必有用　everyone is number one　凡事要不問自己能不能　爭一口氣志向要堅定　當把握機不必等　everyone is number one　
成功的要領在你肯不肯　用最真的心去努力經營　第一名屬於每個人
　　　　　　　　　　　　　　　　　　　　　　　　　　　　（其一）　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　
　　　　　　　　　　　　　　　　　　　　　　　　
　　　　　　　俺乃
南極老壽星
中八仙呂洞賓　同領
母命　來至佛庭　進門已參
皇母容　再問一聲寶寶們各個否安怡
　　　　問聲賢士各個皆歡喜
　呂洞賓大仙慈訓：
　　用雙手殷勤的播種　要一步一步的耕耘　只要一條心　風雨生信心　同修要互牽引　修道中自立自強秉至誠　有一股熱忱　馳騁心感恩　相信自己有志之者事竟成　everyone is number one　只要肯付出沒有不可能　需要勇氣陪伴這一生　當迎向未來的光明　everyone is number one　修辦道都有自己的歷程　苦樂掺雜中無怨尤的心　做一個快樂修道人　不害怕路上多艱辛　一路走來堅持到底的發奮　使命肩承　everyone is number one　凡事要不問自己能不能　爭一口氣志向要堅定　 當把握機會不必等　everyone is number one　成功的要領在你肯不肯　用最真的心去努力經營　第一名屬於每個人　　　　　　　　　　　　　　　　　（其二）
　　　　　　　　　　　　　　　　　　　　調寄：Everyone is nmber one　　　　　　　　　　　　　　　　
　　　　　　　　　　　　　　　　　　

　
　　　　　　　　　　　　　　　　　

　 西元二００九年歲次己丑閏五月廿七日　　　　　　慈訓

　靈隱禪師慈訓：
修辦道都要攀越心中山峰　才能夠遨翔天空　徒兒有不同使命必須完成　共創未來相親相擁　一步一腳印能無盡的攀登　一路修辦多少感動　生命的成長是超越的旅程　一山還比一山更高峰　徒要奔向萬山峻嶺　迎著風雨迎著修道光榮　徒要屹立萬山峻嶺　濟公為徒高興　展露燦爛笑容
　　　　　　　　　　　　　　　　　　　　　　　　　調寄：萬山之巔

　　　　　　　吾乃
靈隱禪師　瘋瘋和尚
中八仙藍采和　領
母諭旨　來至東塵　隱身進入佛庭　早參
母慈容　問聲各位賢士可安好

　藍采和大仙慈訓：
信念始終一　展現己理想志不移　聞過喜修道見賢思齊　踶跂為義勤努力　
　當仁不讓謙遜無驕氣　修辦道彼此分享輝煌的榮譽　給予關懷勇氣與鼓勵　集腋成裘團結秉正義　就算有多大的風雨　腳步要沉穩而不恐懼　仁者無敵　手拉手心連心駕舟楫　要勇敢的面對挑戰的瞬息　不管是輸贏都要感恩歡喜　君子不器　學不厭誨不倦更積極　不同的際遇可創造奇蹟　隨著太陽照耀溫暖大地
　　　　 調寄：信念

中庸其至矣乎民鮮能久矣省思過
知白黜緇循大道施予恩惠無饋求
信誓而旦旦濟弱扶困英漢助末後
大道在細微匡濟仁風迴轉應對策
觀微旦孰營桓公扶正舉賢人明哲
人人向道物阜好合心傾耳聽而獲
安貧之陋宅曲肱匡助幫綺語不說
道之不行也我知之矣中和剛與柔
智者過之愚者不及也互補不足做
望塵而拜迎接客寔寧晉見禮教有
雜草芟刈耕耘做程序清楚定豐碩
道中之俊勿多聲不橫行更正顯卓
仁義說感動攝受魏困假霸術之鬥
囑咐又叮嚀密切效三國趙雲義魄
修道我慢武斷除接受建議互撏撦
道之不明也我知之矣當修正念頭
賢者過之不肖者不及也中道不和
聯邦之虢互盟會友好不煩怎擔憂
緣起緣滅何須問佛法悟弊改不苛
修道旅途龍虎會遵約行韓信功設
開創維艱實踐墾一步一印基礎優
人莫能飲食也鮮能知味也細參究
子曰道其不行矣夫憂道不能耀灼
為人謙牧用於中九鼎一言不輕諾
做道場頗牧之才推展道務勤開拓
國之軍最嚴謹宣威沙漠不怕挫折
心胸莫翦翦精貫白日馳騁旁騖丟
為道不辭起勁奔青天譽響載頌歌
有獎勵有刑罰也丹心誠表不造作
舜其大智也與德化萬民推讓萬德

訓中訓：﹁　　　　　﹂︵先齊其家︶
　　宅曲阜　微旦孰營　桓公匡合
　濟弱扶傾　綺迴漢惠　說感武丁
　俊刈密勿　多士寔寧　晉楚更霸
　趙魏困橫　假途滅虢　踐土會盟
　何遵約法　韓弊煩刑　起翦頗牧
　用軍最精　宣威沙漠　馳譽丹青
